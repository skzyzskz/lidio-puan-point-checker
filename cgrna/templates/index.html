sticky-ip stratejisi nasil kullanilir

bu strateji ile tek proxy uzerinden binlerce farkli ip kullanabilirsin. cok basit:

adim 1: proxy bilgisinde session id olan yere 5 veya daha fazla 0 yaz
adim 2: kaydet tusuna bas
adim 3: hepsi bu. artik her istek farkli ip kullanacak

ornek kullanim:

normal proxy (tek ip):

na.lunaproxy.com:12233:user-abc123:mypass

sticky-ip stratejisi (binlerce ip):

na.lunaproxy.com:12233:user-abc123-sessid-us00000:mypass

bu 5 sifir her istekte random olacak = farkli ip

veya baska yere koy:

na.lunaproxy.com:12233:user-00000-region-us:mypass

veya daha fazla sifir:

na.lunaproxy.com:12233:user-000000000:mypass

9 sifir da olur, hepsi random olacak, sifir degerleri senin proxy saglayicina bagli.

neden kullanmaliyim?

• tek proxy = api seni rate limit eder (yavas)

• sticky-ip = her istek farkli ip = rate limit yok (hizli)

• 300+ kart/saniye hiz icin gerekli

dikkat:

• bu ozellik sadece sticky session destekleyen proxylerde calisir

• lunaproxy gibi servislerde test edildi
